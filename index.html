Travelling Salesman Problem - C++ Research Project

Project Link

Researched on the NP-Complete version of the Travelling Salesman Problem (TSP). Applied Christofides, Kruskal’s, Nearest Neighbour, and Brute Force algorithms to solve several real life specific versions of the TSP. Showed how the methods for each scenario can be modified so the problem can be solved in polynomial time. Improved the previous best runtimes of the algorithms researched by 3 to 10 percent. The project was primarily programmed in C++.
